# Coffee-Shop-Sales

This project presents an interactive Coffee Shop Sales Dashboard built in Microsoft Excel to analyze sales performance, customer purchasing behavior, and product trends. The dashboard transforms raw sales data into meaningful insights using Pivot Tables, Pivot Charts, Slicers, and advanced Excel functions.

The objective of this project is to help coffee shop owners and managers monitor key business metrics, identify top-selling products, understand customer preferences, and make data-driven decisions to improve profitability.

🎯 Objectives

1.Analyze overall sales performance.

2.Track revenue trends over time.

3.Identify top-selling coffee products.

4.Compare sales across different categories.

5.Monitor customer purchasing patterns.

6.Generate actionable business insights through visualization.


🚀 Project Outcomes

1.This dashboard enables stakeholders to:

2.Make informed business decisions.

3.Optimize inventory management.

4.Improve product offerings.

5.Increase revenue through data-driven strategies.
